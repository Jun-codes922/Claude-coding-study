# 할 일 관리 앱 (Todo App) - Product Requirements Document

## 1. 프로젝트 개요

### 1.1 프로젝트명
**Personal Todo Manager** - 개인용 할 일 관리 웹 애플리케이션

### 1.2 목적
사용자가 일상에서 매일 10~20개 정도의 할 일을 간단하고 효율적으로 관리할 수 있는 웹 애플리케이션

### 1.3 타겟 사용자
- 일일 작업 관리가 필요한 개인 사용자
- 간단한 인터페이스를 선호하는 사용자

---

## 2. 핵심 기능 요구사항

### 2.1 할 일 관리 (CRUD)

#### 2.1.1 할 일 추가 (Create)
- 제목이 필수 입력 항목
- 카테고리 선택 (업무 / 개인 / 공부)
- 클릭 버튼으로 신규 할 일 추가

#### 2.1.2 할 일 수정 (Update)
- 기존 할 일의 제목과 카테고리 수정 가능
- 수정 버튼 클릭 시 수정 모드 활성화
- 저장 버튼으로 변경사항 반영

#### 2.1.3 할 일 삭제 (Delete)
- 삭제 버튼 클릭 시 할 일 제거
- 삭제 확인 다이얼로그 표시

#### 2.1.4 완료 체크 (Toggle Status)
- 각 할 일 앞의 체크박스로 완료 상태 표시
- 체크 시 취소선 표시, 체크 해제 시 제거
- 상태 변경 즉시 반영

### 2.2 카테고리 분류
- **3가지 카테고리**: 업무 / 개인 / 공부
- 드롭다운 또는 라디오 버튼으로 선택
- 카테고리별 필터링 표시 가능

### 2.3 진행률 보기
- 전체 할 일 개수 표시
- 완료한 할 일 개수 표시
- 완료율(%) 계산 및 표시
- 예: "완료: 5/12 (41%)"

## 3. 데이터 저장소

### 3.1 저장 방식
- **브라우저 localStorage** 사용
- 새로고침 후에도 데이터 유지
- 별도 백엔드 불필요

### 3.2 데이터 구조

```javascript
// 할 일 객체
{
  id: "unique_id",           // 고유 ID (timestamp 또는 UUID)
  title: "할 일 제목",        // 필수
  category: "업무",          // 업무 / 개인 / 공부
  completed: false,          // 완료 여부
  createdAt: "2026-03-24"   // 생성 날짜
}

// localStorage 저장
localStorage.setItem('todos', JSON.stringify(todoArray));
```

## 4. UI/UX 요구사항

### 4.1 화면 구성
```
┌─────────────────────────────────────┐
│     📝 할 일 관리 (My Todo List)    │
├─────────────────────────────────────┤
│  [할 일 추가] 필터: [전체] [업무] [개인] [공부] │
├─────────────────────────────────────┤
│  ☑ HTML 학습          [업무] [✎] [🗑] │
│  ☐ 보고서 작성         [업무] [✎] [🗑] │
│  ☐ 책 읽기 (30분)      [개인] [✎] [🗑] │
│  ☑ 운동하기            [개인] [✎] [🗑] │
├─────────────────────────────────────┤
│  완료: 2/4 (50%)                    │
└─────────────────────────────────────┘
```

### 4.2 사용성 원칙
- **간단함**: 3-4클릭 이내로 모든 기능 완료
- **직관성**: 명확한 아이콘과 레이블 사용
- **즉각적 반응**: 액션 즉시 반영 (로딩 없음)
- **시각적 피드백**: 완료된 항목은 취소선과 함께 회색 표시

### 4.3 레이아웃
- **반응형 디자인**: 모바일(320px), 태블릿(768px), 데스크톱(1024px) 지원
- **최대 너비**: 800px (데스크톱)
- **패딩/마진**: 16-20px 기본값

### 4.4 색상 스타일
| 요소 | 색상 코드 | 용도 |
|------|----------|------|
| 주색상 | #007AFF | 버튼, 강조 |
| 완료 색상 | #34C759 | 완료 체크, 진행률 바 |
| 삭제 색상 | #FF3B30 | 삭제 버튼 |
| 배경색 | #F5F5F5 | 페이지 배경 |
| 카드 배경 | #FFFFFF | 할 일 항목 |
| 텍스트 | #333333 | 기본 텍스트 |
| 완료 텍스트 | #999999 | 완료된 항목 텍스트 |

## 5. 기술 요구사항

### 5.1 기술 스택
- **마크업**: HTML5
- **스타일**: CSS3 (모던 CSS, Flexbox/Grid)
- **스크립트**: 순수 자바스크립트 (ES6+)
- **저장소**: 브라우저 localStorage
- **프레임워크**: 없음 (Vanilla JS)

### 5.2 성능 목표
- 초기 로딩 시간: 1초 이내
- 할 일 추가/수정/삭제 응답 시간: 즉시
- UI 부드러운 애니메이션 (60fps)

### 5.3 브라우저 호환성
- Chrome (최신 2 버전)
- Firefox (최신 2 버전)
- Safari (최신 2 버전)
- Edge (최신 2 버전)
