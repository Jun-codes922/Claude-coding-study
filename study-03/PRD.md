# 상식 퀴즈 게임 - Product Requirements Document

## 1. 프로젝트 개요

### 1.1 프로젝트명
**Knowledge Quiz Game** - 상식 퀴즈 웹 게임 애플리케이션

### 1.2 목적
사용자가 다양한 주제의 상식 퀴즈를 풀면서 지식을 확장하고, 게임 형식으로 점수를 기록 및 순위를 추적할 수 있는 웹 애플리케이션

### 1.3 타겟 사용자
- 상식 퀴즈에 관심 있는 일반 사용자
- 경쟁심 있고 점수 기록을 좋아하는 사용자
- 다양한 주제로 학습하고 싶은 사용자

## 2. 핵심 기능 요구사항

### 2.1 퀴즈 구성

#### 2.1.1 카테고리
총 **4가지 카테고리** (각 10문제씩, 총 40문제)
- **한국사**: 한국의 역사 및 역사적 인물/사건
- **과학**: 자연과학, 물리, 화학, 생물 등
- **지리**: 세계 지리, 한국 지리, 지형 등
- **일반상식**: 문화, 스포츠, 음악, 영화 등

#### 2.1.2 문제 형식
- **4지선다 객관식** (선택지 A, B, C, D)
- 각 문제마다 1개의 정답
- **난이도 표시**: ⭐ (쉬움) / ⭐⭐ (보통) / ⭐⭐⭐ (어려움)
- **난이도별 필터링**: 사용자가 선택한 난이도만 출제

#### 2.1.3 문제 데이터 구조
```javascript
{
  id: "quiz_001",
  category: "한국사",
  question: "대한민국 건국년도는?",
  difficulty: "easy",  // "easy" / "medium" / "hard"
  options: [
    { id: "A", text: "1945년" },
    { id: "B", text: "1948년" },
    { id: "C", text: "1950년" },
    { id: "D", text: "1952년" }
  ],
  correctAnswer: "B",
  explanation: "대한민국은 1948년 8월 15일 건국되었습니다.",
  hint: "대한민국은 1940년대 후반 남한에서 정부 수립"  // 선택적 힌트
}
```

### 2.2 게임 플레이 기능

#### 2.2.1 게임 시작
- 카테고리 선택 화면에서 원하는 카테고리 선택
- **난이도 선택**: 쉬움 / 보통 / 어려움 (기본값: 전체)
  - 선택한 난이도의 문제만 출제
  - "전체" 모드 옵션으로 모든 난이도 섞여서 출제 가능
- 또는 "전체 퀴즈" 모드로 4가지 카테고리 전체 진행 (40문제)
- 선택 후 "게임 시작" 버튼으로 진행 시작

#### 2.2.2 정답/오답 즉시 피드백
- 선택지 클릭 후 즉시 정답 여부 표시
  - **정답**: 초록색 강조, 체크마크(✓) 표시, "정답입니다!" 메시지
  - **오답**: 빨간색 강조, X 마크(✗) 표시, "오답입니다" 메시지
- 정답 표시와 함께 **해설(설명)** 표시
- "다음 문제" 버튼으로 진행

#### 2.2.3 도움말 기능 (Hint)
- **문제당 1회 제한**: 각 문제마다 도움말 버튼 클릭 가능 (최대 1회)
- **텍스트 힌트**: 문제에 관련된 설명 또는 팁 제공
- **도움말 사용 후에도 정답 100점**: 도움말 사용 여부에 관계없이 정답 시 100점
- **도움말 사용 표시**: 도움말 사용한 문제는 시각적으로 표시

예시:
```
[? 도움말]  ← 클릭하면 "대한민국은 1940년대 후반 남한에서..." 표시
```

#### 2.2.4 진행 상황 표시
- 현재 진행도: "문제 5/10" (카테고리 선택 시) 또는 "5/40" (전체 모드)
- 프로그래스 바로 시각적 진행 상황 표시
- 현재까지의 점수 실시간 표시 (우측 상단)

### 2.3 점수 계산 및 기록

#### 2.3.1 점수 시스템
- **정답 1문제 = 100점**
- 선택하지 않은 경우 = 0점
- 총점 = (정답 개수 × 100점)

#### 2.3.2 최종 점수 화면
게임 종료 후 표시 내용:
```
┌────────────────────────────┐
│    🎉 게임 완료!           │
├────────────────────────────┤
│  카테고리: 한국사           │
│  맞은 문제: 8/10 (80%)      │
│  최종 점수: 800점          │
│  소요 시간: 3분 45초       │
├────────────────────────────┤
│  [신기록! 🏆]              │  ← 높은 점수인 경우
├────────────────────────────┤
│  [다시 풀기] [통계] [홈]   │
└────────────────────────────┘
```

#### 2.3.3 순위 기록
- **localStorage**에 플레이 기록 저장
- 기록 항목: 카테고리, 점수, 정답률, 날짜/시간, 소요 시간
- 기록 데이터 구조:
```javascript
{
  id: "record_001",
  category: "한국사",
  score: 800,
  correctCount: 8,
  totalCount: 10,
  accuracy: 80,
  playedAt: "2026-03-24T14:30:00",
  duration: 225  // 초 단위
}
```

### 2.4 통계 및 순위 기능

#### 2.4.1 개인 통계 보기
- **전체 플레이 횟수**: "총 15회 플레이"
- **카테고리별 최고 점수**: 각 카테고리의 최고 점수 표시
- **평균 정답률**: 모든 게임의 평균 정답률 (%)
- **최근 10개 기록**: 날짜 역순으로 정렬된 플레이 기록 리스트

플레이 기록 리스트 예시:
```
┌─────────────────────────────────────┐
│  📊 나의 통계                       │
├─────────────────────────────────────┤
│  총 플레이: 15회                    │
│  평균 정답률: 76%                   │
├─────────────────────────────────────┤
│  🏆 카테고리별 최고 점수            │
│  한국사: 900점 (2026.03.24)        │
│  과학: 850점 (2026.03.22)          │
│  지리: 800점 (2026.03.20)          │
│  일반상식: 950점 (2026.03.18)      │
├─────────────────────────────────────┤
│  📋 최근 플레이 기록                │
│  1. 03.24 한국사 800/1000 (80%)   │
│  2. 03.23 과학 850/1000 (85%)    │
│  3. 03.22 일반상식 950/1000 (95%)│
│  ... (10개까지 표시)                │
└─────────────────────────────────────┘
```

#### 2.4.2 카테고리별 순위
- 같은 카테고리에서의 플레이 횟수, 평균 점수, 최고 점수 표시
- 예: "한국사: 5회 플레이, 평균 840점, 최고 900점"

### 2.5 오답 반복 학습 기능
- **오답 문제 기록**: 게임 진행 중 틀린 문제 자동 저장
- **오답 재풀기 모드**: 메인 화면에서 "오답 복습" 버튼으로 이전에 틀린 문제들만 다시 풀기
- **진행도 추적**: 오답 문제 중 맞춘 문제 개수 표시
- **오답 메모**: localStorage에 오답 기록 저장 (최근 50개까지)

오답 데이터 구조:
```javascript
{
  quizId: "quiz_001",
  category: "한국사",
  difficulty: "medium",
  question: "대한민국 건국년도는?",
  userAnswer: "A",      // 사용자가 선택한 정답
  correctAnswer: "B",   // 실제 정답
  failedAt: "2026-03-24T14:30:00",  // 틀린 시간
  lastReviewedAt: null  // 마지막 복습 시간 (null이면 아직 복습 안함)
}
```

### 2.6 추가 기능

#### 2.6.1 다시 풀기 (Retry)
- 동일한 카테고리/모드로 새로운 문제 세트로 다시 게임 재개
- 기록은 별도로 저장됨

#### 2.6.2 데이터 초기화
- "모든 데이터 삭제" 옵션 (확인 다이얼로그 필수)
- 플레이 기록 및 설정 모두 삭제

## 3. 데이터 저장소

### 3.1 저장 방식
- **브라우저 localStorage** 사용
- 새로고침 후에도 데이터 유지
- 별도 백엔드 불필요

### 3.2 저장 데이터 항목
```javascript
// 1. 퀴즈 문제 데이터 (정적 데이터)
localStorage.setItem('quizzes', JSON.stringify(quizArray));

// 2. 플레이 기록
localStorage.setItem('playRecords', JSON.stringify(recordsArray));

// 3. 오답 기록
localStorage.setItem('wrongAnswers', JSON.stringify(wrongAnswerArray));

// 4. 사용자 설정
localStorage.setItem('userSettings', JSON.stringify({
  theme: 'light',           // 라이트/다크 모드
  soundEnabled: true,
  lastDifficulty: 'all',    // 마지막 선택한 난이도
  lastCategory: 'korean'    // 마지막 선택한 카테고리
}));
```

## 5. 기술 요구사항

### 5.1 기술 스택
- **HTML5**: Semantic markup, viewport meta tag
- **CSS3**: Custom properties, Flexbox, Grid, transition
- **JavaScript**: ES6+ (프레임워크 없음)
- **저장소**: localStorage

### 5.2 브라우저 호환성
- Chrome, Firefox, Safari, Edge (최신 2버전)

- 선택지 클릭 응답: **즉각적** (밀리초 단위)
- UI 부드러움: **60fps**

## 7. 개발 단계 (예상 일정)

### STEP 1: 기본 구조 + 퀴즈 진행 (2.5시간)
- 메인 홈 화면
- 카테고리 및 난이도 선택
- 퀴즈 진행 화면 (선택지 미표시)
- 도움말 버튼 UI

### STEP 2: 정답/오답 피드백 + 도움말 (2시간)
- 선택지 클릭 처리
- 정답/오답 즉시 표시
- 해설 표시
- 도움말 기능 (텍스트 힌트 표시, 1회 제한)
- 다음 문제 진행

### STEP 3: 점수 계산 및 최종 화면 (1.5시간)
- 점수 계산 로직 (도움말 사용 여부 무관)
- 오답 기록 저장
- 게임 완료 화면
- 결과 표시 (점수, 정답률, 소요 시간)

### STEP 4: 통계 + 오답 복습 기능 (1.5시간)
- 플레이 기록 localStorage 저장
- 오답 기록 저장
- 통계 화면 (카테고리별 최고 점수, 평균 정답률)
- 오답 복습 모드 (이전 오답 문제만 재풀기)
- 플레이 기록 리스트 조회

### STEP 5: UI 다듬기 + 최적화 (1.5시간)
- 반응형 디자인 적용
- 다크 모드 지원
- 난이도별 UI 체계 (아이콘, 색상 구분)
- 성능 최적화
- 전체 테스트

**예상 총 소요시간: 8.5시간**
